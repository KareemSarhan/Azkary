# AI Context – Azkary App

This project uses:
- Kotlin + Jetpack Compose
- MVVM + Repository
- Hilt (Dependency Injection)
- Room (Local DB) + DataStore (Preferences)
- Kotlin Serialization (JSON)
- Compose Navigation
- No Firebase
- Offline-first philosophy
- Islamic content must remain authentic and unchanged

When suggesting changes:
- Prefer simple, explicit logic
- Avoid over-engineering
- Prioritize correctness over abstraction
