# Feature Graphic

The feature graphic is displayed at the top of the app's F-Droid page.

## Requirements

- Dimensions: 1024 x 500 pixels
- Format: PNG or JPG
- Filename: `featureGraphic.png` or `featureGraphic.jpg`
- Max file size: 1 MB

## Design Guidelines

1. **Safe zone**: Keep important content within 769x500 (left side)
   - The right side may be cropped on some devices
2. **Content**: Show app name and key visual element
3. **Style**: Simple, clean design that represents the app
4. **Branding**: Use app colors and style

## Example Layout

```
+------------------------------+
|                              |
|   Azkary                     |
|                              |
|   [Islamic geometric         |
|    pattern or illustration]  |
|                              |
+------------------------------+
```

## Creating the Graphic

Tools you can use:
- Canva (https://www.canva.com)
- Figma (https://figma.com)
- GIMP (free)
- Photoshop

## Notes

- Do not include device frames
- Do not include screenshots within the graphic
- Avoid text-heavy designs
